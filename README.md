# Richard Joseph L. Dela Cruz - Portfolio Website

This is my **personal portfolio website**, showcasing my projects, skills, and contact information.  
It includes links to my GitHub, LinkedIn, resume, and featured projects with images and demos.

## Features
- **Responsive design** using HTML5, CSS3 (Flexbox & Grid).
- **Projects Section**:
  - SCI-HIGH (Educational Roguelike Game) – with live demo link.
  - Dog Catalog Website.
  - Buko OS (built with Cosmo Dev Kit, C#) – includes video demo.
- **Downloadable Resume** (`Portfolio.pdf`).
- **Contact Section** with email and phone number.
- **Social Links** to GitHub and LinkedIn.
